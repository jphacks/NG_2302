# フロントエンド

## React 実行環境セットアップ

### npmのインストール

ターミナルで`./frontend`のディレクトリへ行き、下記コマンドを実行
```
npm install
```

### ライブラリのインストール

```
npm install react-router-dom@6
npm install @mui/material @emotion/react @emotion/styled
npm install react-speech-recognition
npm install axios
npm install react-cookie
```

### 開発サーバー起動

下記コマンドを`.frontend/`で実行

```
npm start
```
